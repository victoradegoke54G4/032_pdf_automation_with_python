# PDF Automation with Python

This project automates PDF cover page operation.  
It’s designed to make PDF handling fast and efficient for everyday use or batch document processing.

---

## Library Used

- **Python 3.8+**  
- **PyPDF2** — for working with PDF files  
- **os** and **shutil** — for file system automation 

---

## Project Structure
```
032_pdf_automation_with_python/
├── Input Files            # Contains files
```
